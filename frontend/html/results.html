Cluster Results

Each card above represents a patient cluster identified by K-Means clustering. The cards display key averaged metrics (Age, BMI, Blood Sugar, Cholesterol, Visits, Medications) for each group, along with a risk badge. Purpose: Provides an at-a-glance clinical profile of each cluster so healthcare providers can quickly understand the characteristics that define each patient group and prioritize care accordingly.

Cluster Interpretation

ClusterProfile DescriptionRisk Level

This table maps each cluster to a human-readable profile description and its assigned risk level. Purpose: Translates raw clustering output into clinical language — helping staff understand which patients fall into low, moderate, high, or critical risk categories. Meaning: Risk labels are derived from the average feature values within each cluster, so higher-risk labels indicate clusters with elevated BMI, blood sugar, cholesterol, blood pressure, visit frequency, and medication counts.

Recommendations

The accordion above provides actionable care recommendations tailored to each risk level. Purpose: Bridges the gap between data analysis and clinical action — giving providers specific guidance on preventive measures, monitoring schedules, and intervention strategies for each risk group. Meaning: Recommendations escalate in urgency and specificity from low-risk (routine checkups) to critical-risk (immediate specialist referrals and intensive monitoring).
